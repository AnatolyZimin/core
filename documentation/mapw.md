NAME
----
mapw - map or unmap window

SYNOPSIS
--------
    mapw [-mut <wid> [wid..]]

        -m : map (show) windows
        -u : unmap (hide) windows
        -t : toggle windows visibility

ENVIRONMENT
-----------
mapw follows the variable `DISPLAY` to set X.org display to act on.
